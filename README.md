# tcf-ai-tutor

A personal project to help me prepare for TCF Canada
while building hands-on experience with multi-agent LLM systems,
voice AI, and LLM observability.

## Why this project
- I'm preparing for TCF Canada
- I want hands-on experience with LangGraph, Whisper, and Langfuse
- Existing tools (OurTCF, PrepMyFrench) focus on content delivery;
  this project focuses on the AI infrastructure layer

## Status
- [ ] Phase 1: Foundation (schema, infra setup)
- [ ] Phase 2: Writing AI Grader
- [ ] Phase 3: Speaking Voice Agent
- [ ] Phase 4: Observability

## Stack (planned)
- LangGraph (multi-agent orchestration)
- Whisper (speech-to-text)
- Langfuse (LLM observability)
- TBD: frontend, database
